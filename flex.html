Section 1 Default flex settings

  • Applying 'display:flex' applies various default behaviours
  • Compare 1.1 normal display with 1.2 flex display
    • Flex removes block behaviour from block elements
    • Flex aligns items in the row direction
    • Flex stretches items in the column direction
  • Flex doesn't wrap by default. Check 1.3 to see `flex-wrap: wrap`
  • Adjust default row alignment. Check 1.4 to see 'justify-content' from 'flex start' to 'space-evenly'
    • Center on wrap? - as items wrap 'space evenly' is applied to each new line.
  • Adjust default column stretch. Check 1.5 to see `align-items: flex-start`
  • Adjust flex direction. 1.6 shows direction: column.
    • Stretch in the row direction?. The default column stretch is actually cross-axis stretch, which has now changed
